# 🤖 Бібліотека агентів для Claude Code

Інтерактивний каталог **232 AI-агентів** для [Claude Code](https://claude.com/claude-code) — з пошуком, фільтрами за категоріями та повними промптами.

## 🔗 Відкрити

**https://iraank.github.io/claude-agents-library/**

## Що всередині

- 🔍 Живий пошук по назві та опису
- 🏷️ 17 категорій із фільтрами (Marketing, Engineering, Sales, Security, Design…)
- 🃏 232 картки — клік відкриває повний промпт агента
- 📱 Адаптивний дизайн (десктоп / мобільний)

## Джерело

Агенти з відкритої бібліотеки [agency-agents](https://github.com/msitarzewski/agency-agents). Встановлені локально в `~/.claude/agents/` і працюють нативно в Claude Code — виклик за іменем у чаті або через Agent tool.

## Структура

| Файл | Призначення |
|------|-------------|
| `index.html` | Сторінка каталогу (бренд-стиль, пошук, модалка) |
| `agents-data.json` | Дані агентів (назва, опис, категорія, промпт) |
